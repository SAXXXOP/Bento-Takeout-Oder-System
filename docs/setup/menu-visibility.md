# メニュー表示制御（管理者 / 閲覧）

## 目的
普段使わない「管理者向けメニュー」を隠して、誤操作を減らす。

---

## 仕組み（概念）
- Script Properties の `MENU_SHOW_*` を参照してメニュー表示/非表示を切り替える。
- 「導入ツール」「監査/保守」系を隠せると運用が楽。

## 使うプロパティ（1つだけ）
- `MENU_SHOW_ADVANCED`
  - `1`：管理者メニューを表示（導入ツール、監査/保守、バックアップ等）
  - `0`：日々の運用メニューのみ表示（通常はこちら推奨）

---

## 運用例
- **通常運用（閲覧/運用担当）**：`MENU_SHOW_ADVANCED=0`
- **管理者作業をする時だけ**：`MENU_SHOW_ADVANCED=1` → 作業が終わったら `0` に戻す

---

## トラブル
- メニューが消えたまま戻らない  
  1) `MENU_SHOW_ADVANCED` の値を確認（0/1）  
  2) キャッシュがある場合は待つ/再読み込み  
  3) それでもダメならメニューの「🔄 メニューを再表示（設定再読込）」を実行
  
ログと実装も参照：`dev/gs-files.md` / `troubleshooting/faq.md`
