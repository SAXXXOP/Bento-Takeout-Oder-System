# メニュー表示制御（MENU_SHOW_*）

## 目的
普段使わないメニューを隠して、誤操作を減らす。

---

## 仕組み（概念）
- Script Properties の `MENU_SHOW_*` を参照してメニュー表示/非表示を切り替える。
- 「導入ツール」「監査/保守」系を隠せると運用が楽。

※ 実際のキー名は実装に合わせてください。

---

## 運用例
- 通常：運用メニューのみ表示
- 管理者のみ：導入ツールや監査ツールを表示（必要な時だけ ON）

---

## トラブル
- メニューが消えたまま戻らない  
  1) `MENU_SHOW_*` の値を確認  
  2) キャッシュがある場合は待つ/再読み込み  
  3) onOpen が走っているか確認

ログと実装も参照：`dev/gs-files.md` / `troubleshooting/faq.md`
