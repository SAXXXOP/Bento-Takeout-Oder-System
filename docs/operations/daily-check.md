# 日々のチェック（締切後〜当日）

## 目的
締切後〜当日朝に、抜け漏れ（要確認・無効・未入力）を潰して当日の作業を安定させる。

---

## 1. 締切後（例：前日20:00以降）にやること

### A) 要確認一覧を見る
- シート：`要確認一覧`
- 観点：
  - 電話番号未入力（フォーム側で必須化推奨）
  - 受け取り日/時刻の不整合
  - 注文内容が読めない/未選択
  - 同一人物の重複や変更の可能性

### B) 注文一覧の当日分をざっと点検
- シート：`注文一覧`
- 観点：
  - 「無効」になっていないか
  - 「★要確認」が残っていないか
  - 変更予約（予約番号）を伴うものが意図通り更新されているか

### C) （任意）通知の確認
- LINE連携を使っている場合、通知が正しく飛んでいるか

---

## 2. 当日朝にやること

### A) 予約札を作成（指定日）
- メニュー：`★予約管理` → `指定日の予約札を作成`
- 詳細：`operations/create-cards.md`

### B) 当日まとめを更新
- メニュー：`★予約管理` → `当日まとめシートを更新`
- 詳細：`operations/production-sheet.md`
