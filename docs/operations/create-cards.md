# 予約札を作成する（指定日）

## 目的
指定日の予約を一覧化し、受け渡しで迷わない「予約札」シートを作成する。

---

## 実行手順
1. スプレッドシートを開く
2. メニュー `★予約管理`
3. `指定日の予約札を作成` を実行
4. 日付入力が出る場合は対象日を指定

---

## 期待結果
- `予約札` シートに指定日の予約が反映される
- 「★要確認」や「無効」の扱いは運用ルールに従う

---

## よくある失敗
- 当日分が出ない：`注文一覧` に当日予約が入っていない / 日付形式が違う
- 途中で止まる：権限不足、または例外

→ `troubleshooting/logs.md`
